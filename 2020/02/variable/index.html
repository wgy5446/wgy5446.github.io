변수

변수는 하나의 값을 저장하기 위한 메모리 공간 자체 or 메모리공간을 식별하기 위해 붙인 이름이다.

변수 여러 개의 값 저장 방법

변수에 여러개의 값을 저장하는 방법에는 여러 개의 변수를 사용한다, 즉, 배열이나 객체와 같은 자료 구조를 사용하여 여러 개의 값을 그룹화하여 하나의 값처럼 사용하면 된다.

식별자

식별자는 어떤 값을 구별하여 식별해낼 수 있는 고유한 이름이며, 값이 아닌 메모리 주소를 기억한다.

변수 선언

변수 선언을 하면 값을 저장하기 위한 메모리 공간을 확보하고 변수 이름과 확보된 메모리 공간의 주소를 연결하여 값을 저장할 수 있도록 준비하는 것이다.
변수 선언 할때는 var,let,const 키워드를 사용한다.

자바스크립트 엔진

자바스크립트 엔진은 변수 이름을 등록하여 자바스크립트 엔진에 변수의 존재를 알리고 값을 저장하기 위한 메모리 공간을 확보하고 암묵적으로 undifined를 할당한다.

변수 호이스팅

변수 선언이 소스 코드가 순차적으로 실행되는 런타임 이전 단계에서 먼저 실행된다는 것을 변수 호이스팅이라 한다.

값 재할당

이미 값이 할당되어 있는 변수에 새로운 값을 또다시 할당하는 것을 말한다.

원시값과 객체의 차이

원시값은 값을 변경 할 수 없고, 객체는 변경 할 수 있다.

가비지 컬렉터

애플리케이션이 할당한 메모리 공간을 주기적으로 검사하여 더 이상 사용되고 있지 않는 메모리를 해체하는 기능을 말한다.
